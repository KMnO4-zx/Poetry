### 诗词生成器

#### 想来看一下运行样例

##### 随机生成一首古诗

```
柳树青枫春水边，
春风相爱起长条。
欲知身外多多限，
莫笑渔时到上庭。
```

##### 续写古诗

```
一江春水向东流，
又带秋山落日边。
闻道千年更惆怅，
未因诗信得知非。
```

##### 藏头诗

```
好岁为吾友，
好分从夏生。
学书曾不废，
习事不辞贫。
天气分玄日，
天河早海心。
向空应有意，
上望日相催。
```
